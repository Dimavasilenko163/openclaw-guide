Что умеет ИИ-агент

💻
Пишет код и создаёт сайты
Лендинги, MiniApp для Telegram, веб-приложения, боты — агент пишет, тестирует и деплоит. Полноценный разработчик в чате.
📝
Контент в твоём стиле
Посты, рассылки, сценарии, статьи — агент знает твой голос бренда и пишет так, как пишешь ты. Не шаблоны, а живые тексты.
💬
Отвечает клиентам 24/7
Подключается к Telegram, WhatsApp, CRM. Отвечает вежливо, уточняет детали, оформляет заявки — пока ты спишь.
🎙
Понимает голос и видео
Расшифровывает голосовые, анализирует изображения, генерирует озвучку через ElevenLabs. Мультимедиа из коробки.
📊
Аналитика и мониторинг
Следит за конкурентами, считает ROAS, парсит данные, строит отчёты. Каждое утро — дайджест в Telegram.
🤝
Команда агентов
Несколько агентов работают вместе: один пишет текст, второй делает картинку, третий публикует. Координатор управляет.

Автоматизация по расписанию
Cron-задачи, напоминания, регулярные проверки — агент работает сам, без твоих команд. Настроил один раз — работает всегда.

Помнит всё
Клиентов, договорённости, историю проектов, твои предпочтения. Не забывает между сессиями — память в файлах навсегда.
🔌
Интеграции с чем угодно
CRM, Google Sheets, Notion, базы данных, любые API — агент подключается через кастомные скиллы. Расширяется без ограничений.

Open source — ваши данные у вас
Код открыт, агент работает на вашем компьютере. Никакие данные не уходят третьим лицам. Полный контроль.

Шаг 1

Установи Node.js

🍎 Mac
🪟 Windows
🐧 Linux
# Установи Homebrew (если нет) /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # Установи Node.js brew install node
3 простых шага:
1. Зайди на nodejs.org
2. Скачай LTS версию
3. Установи как обычную программу
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - sudo apt-get install -y nodejs

Проверь установку:

node --version # → v20 или выше ✅
Шаг 2

Установи OpenClaw

1
Установка
npm install -g openclaw
2
Инициализация
openclaw init
Выбери: имя агента, канал Telegram, модель Claude Sonnet
3
API-ключ
openclaw config set ANTHROPIC_API_KEY sk-ant-...
Ключ получаешь на console.anthropic.com
Шаг 3

Подключи Telegram

1
Создай бота
Открой @BotFather в Telegram → напиши /newbot → придумай имя → скопируй токен
2
Подключи токен
openclaw config set TELEGRAM_BOT_TOKEN 123456:ABC...
3
Запусти!
openclaw start
Открой бота в Telegram → напиши «Привет» → он ответит 🎉
